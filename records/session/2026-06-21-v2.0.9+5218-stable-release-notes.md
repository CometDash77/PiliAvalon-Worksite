# PiliAvalon v2.0.9+5218

这是 PiliAvalon 2.0 系列的当前稳定版本，基于 PiliPlus 2.0.9 上游代码同步，并保留 PiliAvalon 已验证的屏蔽/过滤相关体验。

## 下载说明

请在本页面底部的 **Assets** 中下载适合设备架构的 APK：

- 大多数近年的 Android 手机和平板：`arm64-v8a`
- 较旧的 32 位 Android 设备：`armeabi-v7a`
- Android 模拟器或少数 x86_64 设备：`x86_64`

如果你不确定设备架构，通常优先选择 `arm64-v8a`。

## 本次更新

- 同步到 PiliPlus 2.0.9 上游基线。
- 保留并验证 PiliAvalon 现有屏蔽/过滤相关能力。
- 修复合并后直播间弹出菜单图标颜色导致的构建问题。
- 重新生成依赖锁定文件，确保当前版本可复现构建。

## 安装提示

- 直接覆盖安装即可，不需要先卸载旧版本。
- 如果系统提示签名不一致，请先确认下载的是本页面的 APK，并避免混用其他来源的安装包。
- 覆盖安装依赖相同应用 ID：`com.example.piliplus`。

## 已知注意事项

- 本次稳定版面向 Android APK 发布；iOS、macOS、Windows、Linux 不在本次发布范围内。
- GitHub 自动化已覆盖构建与 Android 模拟器基础运行检查，但不同机型和系统版本仍可能存在个别差异。

## 用户需要做什么

下载对应架构的 APK，直接安装或覆盖安装即可。
